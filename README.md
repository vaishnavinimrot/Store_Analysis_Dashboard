## Store Performance Dashboard

### About Dashboard
This dashboard analyzes store sales and performance data to track revenue, profit, orders, and customer trends. It helps businesses understand store performance and identify growth opportunities.

### Data Source
- SQL Database
- Excel Files

### Features
- Sales and profit analysis
- Region-wise and category-wise performance tracking
- Monthly and yearly sales trends
- Top-performing products and stores
- Interactive filters for better analysis
- Dynamic KPIs and visual reports

### Tools & Technologies
- Power BI
- SQL
- Excel
- DAX
- Power Query
- Data Modeling
