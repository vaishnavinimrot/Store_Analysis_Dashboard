## Store Performance Dashboard

### About Dashboard
This dashboard analyzes store sales and performance data to track revenue, profit, orders, and customer trends. It helps businesses monitor performance, identify trends, and improve decision-making.

### Data Source
- SQL Database
- Excel Files

### Features
- Sales and profit analysis
- Region-wise and category-wise performance tracking
- Monthly and yearly sales trends
- Top-performing products and stores
- Interactive filters and slicers
- Dynamic KPIs and visual reports
- Time intelligence calculations for sales comparison
- Drill-through reports and detailed insights

### Tools & Technologies
- Power BI
- SQL
- Excel
- DAX
- Power Query
- Star Schema Data Modeling
- Data Cleaning & Transformation
- Time Intelligence Functions
